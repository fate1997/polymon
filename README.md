# A unified framework for polymer property prediction
We provide a unified framework for polymer property prediction. This framework is designed to be flexible and easy to use.

## Installation
```bash
pip install polymon
```

## Usage
### Train
```bash
polymon train --help
```

### Merge
```bash
polymon merge --help
```

### Predict
```bash
polymon predict --help
```
