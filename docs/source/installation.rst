Installation
============

You can simply install the package via PyPI:

.. code-block:: none

    pip install polymon

Alternatively, you can install the package from the source code:

.. code-block:: none

    git clone https://github.com/PolyMon/polymon.git
    cd polymon
    pip install -r requirements.txt
    pip install -e .
